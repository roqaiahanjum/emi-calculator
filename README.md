# 🧮 Smart EMI Calculator

A premium, high-performance EMI Calculator built with a modern **Dark Glassmorphism** aesthetic. This tool provides instant financial insights with real-time calculations, visual data representation, and a detailed monthly breakdown.

## 🚀 Features

- **Instant EMI Calculation**: Results update in real-time as you type.
- **Doughnut Chart Visualization**: Powered by Chart.js, showing the percentage split between Principal and Total Interest.
- **Detailed Amortization Schedule**: A monthly breakdown of Principal paid, Interest paid, and the Remaining Balance.
- **Flexible Tenure**: Seamlessly toggle between **Years** and **Months**.
- **Premium UI**: Modern dark-mode design with backdrop-blur effects (Glassmorphism) and smooth animations.
- **Fully Responsive**: Optimized for all devices from 320px mobile screens to large desktop monitors.
- **Edge Case Handling**: Gracefully handles invalid inputs, zeros, or empty fields with descriptive resets.

## 📖 How It Works

### The EMI Formula
The calculator uses the standard Equated Monthly Installment formula:

EMI = [P × R × (1+R)^N] / [(1+R)^N - 1]

Where:
- **P**: Principal Loan Amount
- **R**: Monthly Interest Rate (Annual Rate / 12 / 100)
- **N**: Number of monthly installments

### Amortization Logic
The schedule is generated by iterating through the tenure month-by-month:
1. **Interest for the month** = `Remaining Balance` × `Monthly Interest Rate`.
2. **Principal for the month** = `Monthly EMI` - `Interest for the month`.
3. **New Balance** = `Remaining Balance` - `Principal for the month`.
